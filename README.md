# Skate park sim

A tiny simulation of a skate park, created to test a skate system for a bigger city sim!

## How to

Press play or use Space bar to play/pause the simulation.

## Tech stack

- [Pim art](https://github.com/lovejansson/pim-art)
- [Vite] (https://vite.dev/guide/)
- [Typescript](http://typescriptlang.org/)
